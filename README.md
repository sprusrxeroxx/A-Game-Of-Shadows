# A-Game-Of-Shadows
Ever wanted to play chess without the stress of the clock, or beating an opponent, or stressing about which move combination will produce a masterpiece (in your head anyways) ? A game of shadows is a godot chess game meant to be an intimate experience between two players, a soothing relaxed conversation meant to bond over. Maybe pretentious but ey.
